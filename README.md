# Проект: Путешествие по России

**Описание**

Сайт о путешествии по России. Он рассказывает и показывает необычные места нашей страны. В нем есть как таблица с фотографими мест, так и полезные ссылки.

**Технологии**

* HTML
* CSS
* Figma
* PerfectPixel
